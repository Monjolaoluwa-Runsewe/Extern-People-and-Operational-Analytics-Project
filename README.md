# Extern-People-and-Operational-Analytics-Project
Amazon Employee Experience Analysis
**Overview**
Analyzed employee feedback to identify the primary drivers of dissatisfaction among Amazon warehouse associates and designed an evidence-based intervention to improve retention and workplace experience.
**Problem**
Employee reviews consistently highlighted concerns related to management, scheduling, and compensation, contributing to high turnover among warehouse associates.
**My Role**
I independently:
Conducted research on employee sentiment.
Identified key themes from employee feedback.
Prioritized workforce segments based on operational impact.
Designed a people-first intervention strategy.
Developed implementation recommendations and success metrics.
Created and presented the final presentation.
**Key Recommendation**
A three-layer intervention:
Peer Mentorship
Emotional Intelligence Training for Managers
Manager Review System

supported by an 8-week pilot program.
